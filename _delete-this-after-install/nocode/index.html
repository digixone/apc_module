APC Module Information:

"APC Settings Guide"

  • It is always better to protect your APC PHP Cache login access
  • The default username is "apc" password "apcpassword" you can change this any time.
  • Add more configuration to your "apc.ini"? you can locate this in = /zpanel/modules/apc_module/code/apc.ini.
  • Click this link: http://php.net/manual/en/apc.configuration.php to see options.

NOTES: Do not enable this module to other group such as "User","Reseller". This should only be enabled within Admin Module.

See PHP APC Introductions and Configuration: http://www.php.net/manual/en/intro.apc.php
