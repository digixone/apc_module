Pre-Install check: Install your APC PHP Cache first

"Passwrod Installation Guide"

  • Open you SSH then login as "ROOT"
  • Configure APC PHP-Cache with the following:
  1. Install "NANO" if you don't have it. Type this: yum install nano -y
  2. Change directory to: # cd /etc/zpanel/panel/modules/apc_module/frame
  3. Type: # nano apc.php
  4. Edit the Username and Password.
  5. Save it.
  • To add more configuration on your "apc.ini" located in /etc/php.d/apc.ini.
  • Go click this link: http://php.net/manual/en/apc.configuration.php

NOTES: Do not enable this module to other group such as "User","Reseller". This should only be enabled within Admin Module.

See PHP APC Introductions and Configuration: http://www.php.net/manual/en/intro.apc.php
